# 🍔 BiteBee Food Ordering Application

A full-stack Food Ordering Web Application developed using Spring Boot, Spring MVC, Thymeleaf, Spring Data JPA, Hibernate, and MySQL.

## 📌 Project Overview

BiteBee is a web-based food ordering application where users can browse food items, place orders, and administrators can manage products, restaurants, and customer orders.

This project demonstrates Java Full Stack Development concepts including MVC Architecture, CRUD operations, database integration, and server-side rendering using Thymeleaf.

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/BiteBeeFoodApplication.git
```

Move into the project

```bash
cd BiteBeeFoodApplication
```

Configure MySQL in:

```
src/main/resources/application.properties
```

Run the application using Maven:

```bash
mvn spring-boot:run
```

Or run the `BiteBeeFoodApplication.java` file directly from Eclipse.

---

## 🗄️ Database Configuration

Example configuration:

```properties
spring.datasource.url=jdbc:mysql://localhost:3306/food_app_db2
spring.datasource.username=root
spring.datasource.password=YOUR_PASSWORD
```
